# react-native-styled-datepicker

A Datepicker you can style as you wish

## Installation

```sh
npm install react-native-styled-datepicker
```

## Usage

```js
import { multiply } from 'react-native-styled-datepicker';

// ...

const result = await multiply(3, 7);
```
